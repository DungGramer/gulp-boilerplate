# Gulp Boilerplate

## Feature
+ Minify html, css, js, images, font when build
+ Babel config for js
+ Auto convert for .css, .eot, .tff, .woff font type
+ Hot Reload
+ Clean File before build
+ Prevent pipe breaking caused by errors

## Usage

## Install
```sh
npm install
```
### Run
```sh
npm start
```
### Build
```sh
npm run build
```

### Clean
```sh
npm run clean
```
